# API Documentation for Inventory Management Project

## Endpoints you would need `Authorization: Bearer {`Token`}` for:

### Component
- `GET` component
- `POST` component
- `PUT` component
- `DELETE` component

### Airplane
- `GET` airplane
- `POST` airplane
- `PUT` airplane
- `DELETE` airplane

### AirplaneToComponent
- `GET` AirplaneToComponent
- `POST` AirplaneToComponent
- `DELETE` AirplaneToComponent

### Facility
- `GET` facility
- `POST` facility
- `PUT` facility
- `DELETE` facility

### Customer
- `GET` customer
- `POST` customer
- `DELETE` airplane

### Supplier
- `GET` supplier
- `POST` supplier
- `PUT` supplier
- `DELETE` supplier

### Manager
- `GET` manager
- `POST` manager
- `PUT` manager
- `DELETE` manager
